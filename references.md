# references.md

Curated **links only** — inspiration and source material. We do **not** store
third-party screenshots, CSS, or designs in this repo. Taste is original, merely
*informed by* looking at these.

## Component sources (MIT — safe to adapt and own)
- **shadcn/ui** — https://ui.shadcn.com — the primitives base; designed to be
  copied and owned. Our `Button` is adapted from it.
- **Aceternity UI (free components only)** — https://ui.aceternity.com — MIT
  flair on top of the shadcn base. **Check each component's license before
  pulling; never pull "pro"/paid components.**
- **Radix UI primitives** — https://www.radix-ui.com/primitives (MIT) — headless
  behavior/accessibility under many shadcn components.

## Tokens / type
- **Tailwind CSS** — https://tailwindcss.com
- **Inter (OFL)** — https://rsms.me/inter/ — bundled UI/body/heading family.
- **class-variance-authority** — https://cva.style — variant API used by `Button`.
